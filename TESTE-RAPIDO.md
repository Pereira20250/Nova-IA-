```
❌ "me explica o que é Python"  (Muito longo)
```

## 🔧 Verificação Técnica

Se algo não funcionar, verifique:

### 1. Internet Ativa?
```
Abra Google no navegador:
✅ SIM, pode navegar → Internet OK
❌ NÃO consegue → Vérif. WiFi/dados
```

### 2. Microfone?
```
Teste em google.docs.com:
✅ Reconhece voz → Microfone OK
❌ Não reconhece → Aumentar volume/permissão
```

### 3. Navegador Compatível?
```
Use um destes:
✅ Chrome (versão 25+)
✅ Edge (versão 18+)
✅ Firefox (versão 25+)
✅ Safari (versão 14.1+)

❌ Internet Explorer → NÃO suportado
❌ Navegadores muito antigos → Atualizar
```

### 4. Console Errors?
```
Pressione F12 → Aba "Console"
✅ Sem erros → Tudo bem
❌ Com erros em vermelho → Ver abaixo
```

## 🆘 Se Não Funcionar...

### Problema: "Desculpa, não consegui buscar"

```
Solução:
1. Verificar internet (abra www.google.com)
2. Reiniciar navegador
3. Limpar cache (Ctrl+Shift+Del)
4. Tentar novamente
```

### Problema: "Não encontrou na Wikipedia"

```
É normal! Algumas perguntas muito específicas
não existem em português na Wikipedia.

Solução:
1. Reformule a pergunta
2. Tente em inglês simples
3. Use uma pergunta mais geral

Exemplo:
❌ "O que é espectrometria de massa?"
✅ "O que é espectrometria?" (mais geral)
```

### Problema: "Clima de [cidade] é inválido"

```
A cidade não foi encontrada no banco de dados.

Solução:
1. Confirme o nome da cidade
2. Tente com a grafia correta
3. Tente uma cidade maior

Exemplo:
❌ "Clima em N. Hamburgo"
✅ "Clima em Hamburg" ou "Clima em Hamburgo"
```

### Problema: Demora para responder

```
As buscas levam 1-3 segundos.

Se for mais que isso:
1. Verificar velocidade da Internet (speed.com)
2. Reiniciar roteador WiFi
3. Fechar abas desnecessárias
```

```

7. Encontra artigo
8. Formata resposta

   desoxirribonucleico é uma molécula que contém as 
   instruções genéticas para a vida..."

11. Histórico atualizado
```
